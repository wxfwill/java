# 企业权限与业务管理平台

该项目是全年主线项目，完成阶段三后开始创建实际后端模块。

## 目标能力

- 用户、角色、菜单、组织机构
- 菜单、按钮、接口和数据权限
- Access Token 与 Refresh Token
- 操作日志、登录日志、文件、字典和配置
- Redis、任务、导入导出
- 自动化测试、Compose、Nginx、Actuator 和回滚

## 进入条件

- 已通过阶段一、阶段二验收。
- 能独立设计权限数据库和 REST API。
- 能手写 SQL、事务和基础测试。

## 建立顺序

1. 先建立 API 契约和数据库迁移。
2. 再实现模块化单体和最小业务模块。
3. 安全、缓存、部署和可观测性按阶段逐步引入。
